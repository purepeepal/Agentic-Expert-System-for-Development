# 📚 KNOWLEDGE VAULT INDEX

> Master index of all stored knowledge, decisions, directives, and context. Updated automatically by the `/store-knowledge` workflow.

## How This Works

Every piece of information the user/developer shares is stored in this vault. Entries are classified, stored in the appropriate subdirectory, and indexed here for fast retrieval. When this system is dropped into a new project, the vault starts fresh — previous entries are archived.
